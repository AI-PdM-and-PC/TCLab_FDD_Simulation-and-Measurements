# TCLab FDD Simulation and Measurements

This repository contains German teaching materials for Fault Detection and Diagnosis (FDD) with the Temperature Control Lab (TCLab). The focus is on simulated and real TCLab time series, the comparison of normal operation and faulty behavior, and data-driven methods for anomaly and novelty detection.

The notebooks are designed for teaching in process automation, applied AI, data science, and predictive maintenance. They show step by step how a transparent FDD workflow can be developed from process measurement data: from simple baseline methods and SPC control charts to novelty detection models on simulated and real data.

## Target audience

The material is intended for students in process automation, applied AI, data science, and predictive maintenance.

Basic knowledge of Python, Jupyter Notebooks, data analysis, and fundamental control engineering is helpful. The TCLab serves as a compact demonstrator for thermal process dynamics, measurement data analysis, and fault detection.

## Contents

The repository covers:

- TCLab measurements and TCLab simulations
- normal operation and fault experiments
- residual generation as a simple baseline for fault detection
- Shewhart, CUSUM, and EWMA control charts
- generation of structured NOC datasets
- run-wise splitting to avoid data leakage
- windowing for time-series models
- PCA, Isolation Forest, and One-Class SVM
- shallow autoencoder for novelty detection
- controlled tests with injected anomalies
- transfer of simulation models to real TCLab data
- threshold recalibration and retraining on real normal-operation data
